将webm文件转换为.mp4格式

brew install ffmpeg

ffmpeg -i input.webm -c:v libx264 -c:a aac -strict experimental -b:a 192k output.mp4

（该参数对音频使用192 kBit / s恒定比特率）

对于音频，静态构建不支持libfdk-aa。如果你支持它，则应改用：ffmpeg -i input.webm -c:v libx264 -c:a libfdk_aac output.mp4

chsh修改root密码

root用户下使用chsh，若错误修改shell路径（比如改为haha），root用户的shell仍会被强制修改（如 改为haha）。

这样，之后便不能再切换至root用户。

解决方案：

cd /etc/passwd

找到root用户一行（如　root:x:0:0:root:/root:haha）
